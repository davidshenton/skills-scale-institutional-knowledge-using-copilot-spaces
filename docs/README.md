# OctoAcme Project Management Documentation Hub

Welcome to the OctoAcme project management documentation! This README introduces our unified approach to project delivery, outlines key principles, and links to standardized process guides for every project phase.

## Overview: OctoAcme Project Management Processes

- **Customer-first:** Prioritize customer value and usability in all work.
- **Iterative delivery:** Deliver in small, testable increments.
- **Clear roles & artifacts:** Each project has named leads and required docs.
- **Measurable impact:** Use data and evidence to guide decisions and improvements.

OctoAcme's project management lifecycle covers: initiation, planning, execution, release, retrospectives, and risk management. Process artifacts and templates are maintained in this repo, supporting consistent, repeatable project work for everyone.
